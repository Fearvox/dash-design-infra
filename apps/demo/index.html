dash-design-infra

v0.0.1 · Bun

Fixed-canvas editorial design for agents that ship

Design tokens, type scales, baseline metrics, constraint layout, p5.js motion grammar, and print-grade PDF export — all on a single Bun stack. No browser UI overhead; just the boring machinery that makes designed output survive contact with reality.

$ bun run measure:check -- page.html
✓ Fits. 1684×1191

$ bun run print:render -- page.html out.pdf
✓ 7 pages. out.pdf

$ bun run hackathon:score
MAXXED 234/234

$ bun daemon:start
listening on 127.0.0.1:3845

@dash/tokens

Color, typography, spacing, page geometry, motion — single-source design truth.

@dash/scale

Type & spacing scales via utopia-core with φ split-ratio computation.

@dash/metrics

Baseline-safe text metrics via capsize CSS. No magic numbers.

@dash/kami

Warm editorial preset: paper canvas, ink-blue accent, serif hierarchy.

@dash/measure

Playwright overflow checks. CI gate for fixed-canvas pages.

@dash/layout

Cassowary constraint solver. Hard rules like "footer never clips."

@dash/print

HTML→PDF via paged.js + Chromium. Screen preview = print output.

@dash/p5-motion

Reusable p5.js motion presets: Electric Archive, Weather Report.
