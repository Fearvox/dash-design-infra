# Creator Contact-sheet QA Gate

## Contact-sheet QA gate

This slice turns the retained `creator-motion-storyboard-route` into a repeatable QA artifact. It does not commit generated video, screenshots, or model output. It writes a synthetic contact sheet to `.artifacts/creator-motion-contact-sheet.html`, which stays ignored by git.

## Ignored artifact

`.artifacts` is ignored in `.gitignore`. The contact sheet is regenerated by command, not committed. This keeps proof review fast without leaking raw generated media or private render paths.

## Public boundary

Allowed:

- frame timing and motion copy from `examples/creator-motion-storyboard.json`;
- synthetic HTML contact sheet generated under `.artifacts`;
- browser geometry / visual QA against that ignored artifact.

Blocked:

- raw generated video;
- image/video model output committed to git;
- private prompts;
- local absolute source media paths;
- API keys, cookies, account screenshots;
- client copy or unreleased brand assets.

## Verification

```bash
bun creator:contact-sheet-check
bun measure:check -- .artifacts/creator-motion-contact-sheet.html
bun print:render -- .artifacts/creator-motion-contact-sheet.html /tmp/dash-creator-motion-contact-sheet.pdf --canvas=1684x1191
bun security:scan
bun hackathon:score
```
