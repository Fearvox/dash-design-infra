# p5.js Use Cases

`@dash/p5-motion` turns visual experiments into named motion grammar. The goal is not to ship every sketch. The goal is to preserve the reusable part so a future poster, report, or browser artifact can start from a known pattern.

| Usecase | Preview | Good for |
|---|---|---|
| [Electric Archive](./electric-archive.md) | <img src="./assets/electric-archive.png" width="180" alt="Electric Archive preview" /> | archive memory, retrieval traces, handoff evidence, electric blue signal fields |
| [Memory Weather Report](./weather-report.md) | <img src="./assets/weather-report.png" width="180" alt="Memory Weather Report preview" /> | pressure systems, risk forecast, benchmark weather, evidence density |

## Boundary

The screenshots here are cropped public previews. Raw source videos, internal lab paths, and private project content stay outside this repository.
