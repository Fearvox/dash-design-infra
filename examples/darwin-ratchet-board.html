Decision rule

✓
New score must be strictly higher than the baseline.
✓
Visual claims need rendered evidence, not chat confidence.
✓
Public references become grammar, never copied screenshots.
✓
Failed experiments leave a clean revert trail.
01 / VET
Review before install

Unknown skills are references until every file and source→sink path is inspected.

02 / EXTRACT
Keep the pattern, not the asset

Darwin contributes the evaluation loop. DASH supplies its own layout, tokens, and proof path.

03 / PROVE
Browser truth beats intent

Measure the HTML, render the PDF, scan the public boundary, then claim quality.

04 / RATchet
One slice per loop

Small commits, strict gates, PR, CI, merge. No broad heroic rewrites.
