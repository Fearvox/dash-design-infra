Remember the idea before the tools mutate it.

A capsule stores intent, memory, surface, grammar, inputs, tool path, proof, and remix rule. Any frontier visual tool can plug in if it respects that boundary.

◈
Minimal useful unit

Not a CMS, not a gallery, not another prompt dump. One compact creative memory card that can become a poster, board, deck, short, PDF, TouchDesigner patch, or skill.

01 / Brief
Start from creator memory

02 / Render
Choose the smallest path

Use HTML if stillness works, p5 if motion grammar matters, video tools only when time is essential.

03 / Proof
Reality beats taste

Measure layout, render export, scan public boundary, and inspect the actual browser artifact.

04 / Remix
Keep the seed portable

Future tools may mutate the surface, but the capsule preserves identity and constraints.
